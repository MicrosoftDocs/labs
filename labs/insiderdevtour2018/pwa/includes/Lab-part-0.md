# What will this lab cover

This lab will explain how to build a Progressive Web App (PWA) using a simple web application created with ASP.NET Core. You will learn how to use a tool that generates the necessary files to convert your web application into a Progressive Web Aplication.

## Prerequisites

Here are the prerequisites and steps required to set up your computer. After completing this section you will have a working environment, ready for the other steps.


- You will need the Windows 10 April 2018 update. The Windows April Update is required to be able to use Windows Timeline. Download the update from the [software download](https://www.microsoft.com/en-us/software-download/windows10) page.

- Install or update to [Visual Studio 2017](https://developer.microsoft.com/windows/downloads).

-  An Azure account is required to use Azure services. If you don't have an Azure account, you can create one for free  [Azure](https://azure.microsoft.com/en-us/free/). For more information, follow this small tutorial: [How to create up a new Microsoft Azure account.](https://www.acronis.com/en-us/articles/create-microsoft-azure-account/)
